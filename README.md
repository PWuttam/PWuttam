## こんにちは、ウッタムです

清掃会社で品質管理を担当しながら、現場業務を改善するためのWebアプリ・自動化ツールを開発しています。

紙・Excel・手作業・属人化した連絡を、少しずつ「確認しやすい」「抜け漏れに気づける」「現場で使える」仕組みに置き換えています。将来的には、業務理解を強みにしたフルスタックエンジニアとして、完全リモートで働くことを目指しています。

---

## What I Build

- 清掃会社向けの予定通知・巡回管理・スケジュール管理ツール
- Python / Google Apps Script を使った業務自動化
- Next.js / TypeScript を使った業務アプリのプロトタイプ
- 現場で使われるデータを整理し、運用しやすい仕組みに変えること

---

## Selected Projects

現在取り組んでいる代表的なプロジェクトです。

- `notifier`: 清掃予定の確認と担当者への通知を支援する自動化ツール
- `schedule-manager`: 清掃スケジュール管理をより使いやすくするためのWebアプリ
- `patrol-app`: 物件巡回の記録・確認をスマートフォンで行うためのアプリ
- `cleaning-scheduler-demo`: 定期清掃の月間予定作成を支援するデモアプリ

どれも、現場の作業を止めずに少しずつ改善できることを重視して開発しています。
